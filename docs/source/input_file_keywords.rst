Input File Keywords
===================

Two scripts use input files in **Nanoworks**:

* ``dftsolve`` - main DFT calculations
* ``mdsolve`` - molecular dynamics (MD) calculations
* ``mlsolve`` - machine learning calculations

The keyword documentation is split into separate pages for clarity.
